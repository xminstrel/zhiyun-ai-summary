# 课堂总结提示词设计

这个项目默认让 AI 输出一份可以直接复习的 Markdown 学习资料。核心原则：

1. 不编造老师没有讲过的内容。
2. 对语音识别文本中的口头语、重复、轻微错字做合理清洗。
3. 信息不足时标记为“需要回看确认”。
4. 输出必须同时覆盖课堂概览、课程主线、核心知识点、重点难点、学习大纲、复习建议和待确认问题。

可以按课程类型继续扩展：

- 数学/物理：增加“公式推导”“典型题型”“条件与结论”。
- 编程课程：增加“代码步骤”“接口说明”“常见报错”。
- 文史哲课程：增加“人物/概念关系”“观点对比”“论证结构”。
- 实验课程：增加“实验步骤”“仪器材料”“数据处理方法”。
